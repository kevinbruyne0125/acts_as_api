acts_as_api is a gem that aims to make the construction of JSON and XML responses in rails 3 easy and fun.

Therefore it attaches a couple of helper methods to active record and the action controller base classes.

acts_as_api uses the default serializers of your rails app and doesn‘t force you into more dependencies.
